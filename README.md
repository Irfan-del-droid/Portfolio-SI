# Shek Irfan S — Portfolio

Cinematic 3D AI developer portfolio built with React, Three.js, and Tailwind CSS. Featuring an immersive AI brain animation, holographic glassmorphic UI, and bento-grid layouts.

🌐 **Live**: _coming soon_
👤 **Author**: [Shek Irfan S](https://www.linkedin.com/in/shek-irfan-s-b7422133a)
🐙 **GitHub**: [@Irfan-del-droid](https://github.com/Irfan-del-droid)

## Tech Stack

- **React 18** + Vite
- **Three.js** / `@react-three/fiber` / `@react-three/drei` — 3D AI brain & particles
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animations
- **GSAP** — scroll-triggered timeline animations

## Features

- 🧠 Interactive 3D AI brain with lightning, particles, and orbiting rings
- 🎨 Cyberpunk glassmorphic design with cyan/purple/pink neon palette
- 🪟 Bento-grid layouts for Hero, About, Skills, Projects, Experience, Contact
- 📱 Fully responsive
- ⚡ Real tech logos (Python, React, FastAPI, MongoDB, Postgres, Docker, etc.)
- 📥 Downloadable CV
- 💌 Contact form

## Run Locally

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
npm run build
```

Output: `dist/`
